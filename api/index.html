Hi! Welcome to my heroku test app!

There's a hello world test endpoint at GET /api/parse

There's also a recipe ingredient scraper at POST /api/parse

To get started with the scraper, include a raw json object: { "url": "https://www.budgetbytes.com/summer-gazpacho/" } in the req.body and watch the magic happen!
